# 💳 Credit Score Prediction using Machine Learning

## 📌 Overview

This project focuses on predicting an individual's creditworthiness based on financial and personal data using Machine Learning techniques.

The goal is to classify whether a person has a **good or bad credit score**, which is useful for banks and financial institutions in decision-making.

---

## 🎯 Objective

To build a classification model that predicts credit score categories based on user financial data.

---

## ⚙️ Technologies Used

* Python 🐍
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📊 Dataset

The dataset contains financial and behavioral attributes such as:

* Age
* Income
* Loan Amount
* Credit History
* Number of Credit Cards
* Payment History
* Target (Credit Score: Good / Bad)

⚠️ *Note: Dataset may be synthetic or preprocessed for learning purposes.*

---

## 🧠 Machine Learning Models Used

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

---

## 🔄 Workflow

1. Data Collection / Loading Dataset
2. Data Cleaning & Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Visualization

---

## 📈 Model Evaluation

Models were evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report

---

## 📊 Visualizations

* Confusion Matrix Heatmap
* Feature Importance Plot
* Correlation Heatmap

---

## 🏆 Results

* Random Forest achieved the best performance among all models.
* Important factors affecting credit score:

  * Payment History
  * Income
  * Credit Utilization
